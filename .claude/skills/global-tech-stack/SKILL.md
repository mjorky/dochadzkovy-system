## When to use this skill

- When implementing backend features with NestJS, GraphQL, and TypeScript
- When building frontend features with Next.js 14+ App Router and React
- When working with the GraphQL API (queries, mutations, subscriptions) instead of REST
- When using Prisma Client or Drizzle ORM for database operations with PostgreSQL
- When styling components with TailwindCSS and shadcn/ui component library
- When implementing forms using React Hook Form with Zod validation
- When managing state with Zustand instead of Redux or other state libraries
- When writing tests with Jest (backend) or Vitest + React Testing Library (frontend)
- When making architectural decisions that need to align with the tech stack
- When evaluating new libraries or tools for compatibility with existing stack
- When implementing real-time features using GraphQL Subscriptions over WebSocket
- When setting up or configuring development, testing, or deployment environments
